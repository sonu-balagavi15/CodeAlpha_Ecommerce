# CodeAlpha E-Commerce Store

## Description
A Django-based E-Commerce web application developed as part of the CodeAlpha Internship.

## Features
- User Registration & Login
- Product Management
- Product Search
- Categories
- Product Details
- Wishlist
- Shopping Cart
- Checkout
- Order History
- Admin Panel

## Tech Stack
- Python
- Django
- SQLite
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
